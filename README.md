# Rock Paper Scissors – Terminal Edition

A tiny, Python game that pits you against the computer in a classic Rock-Paper-Scissors game.

# Features
1. Picks a random move for the computer (`r`, `p`, or `s`)
2. Asks you for your move
3. Shows the computer’s choice with emojis
4. Announces the winner (or a draw) based on the traditional rules
5. Exits cleanly if you mistype your choice

# Usage
1. Run:
   ```bash

   python rock_paper_scissor.py
